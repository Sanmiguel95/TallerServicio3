### Software a Instalar ⚙️

#### Ejecución Local 💻

Seguir los siguientes pasos:

```
* source .venv/bin/activate
* uvicorn main:app --reload
```

#### Probar localmente 🔎
Ejecución de pruebas unitarias:
```
* pytest
```

Ejecución del site local: 
```
* localhost:8000
```

Swagger
```
* localhost:8000/docs
```
